# BlinkAI: AI-Powered Communication System Using Eye Blinks

A web-based accessibility tool that translates intentional blinks into text using OpenCV, MediaPipe, and Morse code, enabling hands-free interaction with AI.

## Tech Stack
- Frontend: React, TailwindCSS, Socket.IO
- Backend: Flask, Flask-SocketIO, OpenCV, MediaPipe
- AI: Mistral API (planned integration)
